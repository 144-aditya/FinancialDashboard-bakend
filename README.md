# Finance Dashboard Backend

A production-ready backend system for managing personal/finance records with role-based access control.

## Features
- 🔐 JWT Authentication & Authorization
- 👥 Role-based Access Control (ADMIN, ANALYST, VIEWER)
- 💰 Financial Records CRUD Operations
- 📊 Dashboard Analytics (Income, Expenses, Net Balance)
- 📈 Category-wise & Monthly Summary
- 🗄️ H2 Database with JPA/Hibernate
- 📚 Swagger/OpenAPI Documentation
- 🎨 Role-based UI for testing

## Tech Stack
- Java 17
- Spring Boot 3.1.5
- Spring Security with JWT
- Spring Data JPA
- H2 Database
- Maven
